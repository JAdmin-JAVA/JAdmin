工具方法

系统内置了一些工具类，包括数据加解密、数据词典、数据缓存、页面静态化、网络连接、文件操作、excel操作、日期与时间、json等工具类。

加解密工具类 com.jadmin.modules.util.encode.Encode：

返回类型 方法及说明
java.lang.String encode(java.lang.String content, boolean isDecode)
数据加密
参数：
content - 要加密的内容
isDecode - 是否可以解密
返回：加密后的内容
java.lang.String decode(java.lang.String encodeContent)
数据解密
参数：
encodeContent - 加密后的内容
返回：解密后的内容
void setKey(long enKey)
设置可逆加密算法DES的密钥，默认密钥为1231234234l
参数：
enKey - 新的密钥

数据词典工具类 com.jadmin.modules.util.DictinfoUtils：

返回类型 方法及说明
com.jadmin.vo.entity.base.DictkindVO getDictkindById(java.lang.String id, javax.servlet.http.HttpServletRequest request)
給据数据词典的id主键，获取数据词典DictkindVO对象
参数：
id - 数据词典的id主键
request - request请求
返回：数据词典DictkindVO对象
com.jadmin.vo.entity.base.DictkindVO getDictkindByCode(java.lang.String code, javax.servlet.http.HttpServletRequest request)
給据数据词典的code，获取数据词典DictkindVO对象
参数：
code - 数据词典的code
request - request请求
返回：数据词典DictkindVO对象
java.lang.String getDictkindNameBykey(java.lang.String code, java.lang.String key, javax.servlet.http.HttpServletRequest request)
給据code和key，获取name，比如給据sex和1，获取男
参数：
code - 数据词典的code
key - 数据词典的key
request - request请求
返回：name
java.lang.String getDictkindKeyByName(java.lang.String code, java.lang.String name, javax.servlet.http.HttpServletRequest request)
給据code和name，获取key，比如給据sex和男，获取1
参数：
code - 数据词典的code
name - 数据词典的name
request - request请求
返回：key
