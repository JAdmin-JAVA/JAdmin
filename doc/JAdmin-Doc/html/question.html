常见问题

本篇将主要讲解使用过程中普遍遇到的“问题”，这些问题并非是BUG，通常是需要我们自己去注意的一些点。（会结合用户反馈持续补充）
1、后台模板用的哪款？为何不用layui？

layuiAdmin是 layui 出品的一款非常优秀的后台模板，但是需要授权使用，为了不必要的纠纷，JAdmin默认使用了同样优秀的H-ui.admin作为后台模板。 JAdmin本身是开源系统，您可以随意修改后台模板。

2、不登录就能访问后台任意页面？

为方便开发，当项目为调试模式时，访问后台页面后，自动用admin用户登录。将application.yml中的debug修改为false，关闭该功能。 项目正常上线时，请务必关闭调试模式。

#是否开启调试模式
				debug: true
3、一修改代码，系统就自动重启？

系统默认开启了热部署，可以将pom.xml的以下配置注释掉，就可以关闭热部署。


				 
				    org.springframework.boot 
				    spring-boot-devtools 
				    runtime
